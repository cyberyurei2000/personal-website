About this website

I made this website as a small hobby and passing time project to not make my web development knowledge go to waste (ironically, I hate web development! I still have trauma of node.js and react that I got from college).

I started this project after discovering the Melonland forum and seeing some content on Neocities at the end of 2023, it was based on various quirky websites I've come across over the years.

Tecnical details

This website was developed with pure html, css and a minimum amount of javascript without any frameworks, trackers or node.js junk. Javascript is only used for displaying silly fortunes and for the useless clock, and also for the easter eggs.

Although it was created with a focus on the desktop, most of the site is responsive to not exclude users on mobile devices. Even so, I still recommend viewing it on a desktop or any device with a larger screen.

And it's best viewed on Chromium browsers, it can also been viewed on Firefox but has some inacurracies. Safari/WebKit is not supported.
