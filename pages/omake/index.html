Quick terminal commands

This is a collection of quick terminal commands that I put here to share with other people, and also for me to to take a look at if I forget.

Disclaimer

Run any command below at your own risk! Never blindly type commands that you found on the internet without making your own research first.

$ = Normal　|　# = Administrator/Root

Get the best video quality (yt-dlp)
$　yt-dlp -f ”bv+ba” YOUTUBE_URL
Get the best audio quality (yt-dlp)
$　yt-dlp -f ”ba” -x "YOUTUBE_URL"
Increase volume of a media file (ffmpeg)
$　ffmpeg -i ”MEDIA.in” -fa volume=4.0 -c copy ”MEDIA.out”
Trim a media file (ffmpeg)
$　ffmpeg -i "MEDIA.in" -ss 00:00:00 -to 00:00:00 -c copy "MEDIA.out"
Flip image horizontally (imagemagick)
$　convert "IMAGE.in" -flop "IMAGE.out"
Flip image vertically (imagemagick)
$　convert "IMAGE.in" -flip "IMAGE.out"
Extract a archive file (7zip)
$　7z x "ARCHIVE.ext"
Get all devices IP on a network (nmap)
#　nmap -sn 000.000.00.000/00
Setup with a local account (Windows)

Press Shift + F10, then:

#>　OOBE\BYPASSNRO
Get all used network ports (Linux)
#　ss -tulpn | grep LISTEN
Detach a command from the terminal (Linux)
$　COMMAND & disown
Force logout (Linux)
$　pkill -KILL -u $USER
Reboot only the userspace (Linux)
#　systemctl soft-reboot
