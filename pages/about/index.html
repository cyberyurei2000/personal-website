About this website

I made this website as a small hobby and passing time project to not make my web development knowledge go to waste (ironically I hate web development, I still have node.js and react trauma from college).

I started this project after discovering the Melonland forum and seeing some content on Neocities at the end of 2023. I based it on various quirky sites I've come across over the years.

Technical details

This website was made with pure html, css and a minimum amount of javascript without any frameworks, trackers or node.js junk. Javascript is only used for displaying silly fortunes and for the useless clock in the second header bar, and also for the easter eggs.

Although it was created with a focus on the desktop, most of the site is responsive to not exclude users on mobile devices. Even so, I still recommend viewing it on a desktop or any device with a larger screen.

And it should work perfectly in any browser based on Chromium or Firefox, Safari/WebKit is not supported.
